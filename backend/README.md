# My Express Backend

A simple Node.js backend server using Express.

## How to run this project

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   Go to http://localhost:3000

## Built With

- Node.js
- Express.js
- Nodemon (for development)
